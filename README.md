 # PowerBI-business-dashboard

##  Project Overview

This project showcases an interactive **Superstore Dashboard** developed to analyze and visualize key business metrics including sales, profit, customer trends, and regional performance. The dashboard transforms raw retail data into meaningful insights that support data-driven business decisions.

The project focuses on identifying:

* High-performing regions and categories
* Profit and loss trends
* Sales distribution across segments
* Customer purchasing patterns
* Shipping and operational insights
